GET : Utilisée pour demander des données à partir d'un serveur. Les paramètres sont généralement inclus dans l'URL sous forme de chaîne de requête.
	Utilisation fréquente pour récupérer des informations (lecture).

POST : Utilisée pour envoyer des données au serveur, souvent dans le corps de la requête. Cela peut inclure des formulaires, des fichiers, etc.
	Utilisation courante pour envoyer des données (écriture).
	
Pour toutes les requetes : path ->> indique au serveur quelle ressource doit etre traitee 

Format requete :

	- Ligne de requete fini par \r\n
	- Chaque ligne den tete separer par \r\n puis finis dune ligne vide 
	
ex : echo -e "GET /index.html HTTP/1.1\r\nHost: localhost\r\nConnection: close\r\n\r\n" | nc localhost 8080
